## Getting Started

To get a local copy up and running follow these steps.

### Prerequisites

- npm

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo

```sh
git clone https://github.com/github_username/repo.git
```

2. Install NPM packages

```sh
npm install
```

3. Run the project

```sh
npm start
```
